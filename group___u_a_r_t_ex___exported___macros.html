◆ UART_MASK_COMPUTATION

#define UART_MASK_COMPUTATION (   __HANDLE__)

Reports the UART mask to apply to retrieve the received data according to the word length and to the parity bits activation.

If PCE = 1, the parity bit is not included in the data extracted by the reception API(). This masking operation is not carried out in the case of DMA transfers.

Parameters
<strong>HANDLE</strong>specifies the UART Handle
Return values
maskto apply to UART RDR register value.
